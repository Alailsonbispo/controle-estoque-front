# 📦 ESTOQUE-PRO | Dashboard & Gestão ( Projeto Criado com ajuda de IA, visando conhecimento de Java, Spring, Banco de dados)

Sistema Fullstack desenvolvido para controle de patrimônio e estoque, integrando um backend robusto em Java com um frontend moderno e performático.

> 💡 **Nota:** Este é o repositório do **[FrontEnd]**. Para ver o ecossistema completo, acesse o repositório do **[BackEnd]** aqui: [[https://github.com/Alailsonbispo/controle-estoque**]].

## 🚀 Tecnologias Utilizadas
- **Frontend:** Next.js 16, Tailwind CSS, Axios.
- **Backend:** Java 17, Spring Boot, Spring Data JPA.
- **Relatórios:** jsPDF e jsPDF-autotable.

## ✨ Funcionalidades
- 📊 **Dashboard de Patrimônio:** Cálculo em tempo real do valor total investido no estoque.
- 🔍 **Busca Inteligente:** Filtro instantâneo por nome ou categoria de produto.
- 📄 **Exportação PDF:** Geração de relatórios profissionais com cabeçalho personalizado de Salvador/BA.
- ⚠️ **Gestão de Reposição:** Alertas automáticos para itens abaixo do estoque mínimo.
